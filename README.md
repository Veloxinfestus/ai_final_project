# Study Planner — AI‑Assisted Weekly Scheduling Tool

## Project Purpose
This project is a weekly study planner designed to help students organize their study time based on the tasks they need to complete and the hours they have available each day. The goal is to help students manage workload, reduce stress, and build consistent study habits by generating a structured weekly plan.

## Final Goal
The final version of this project will be a web/desktop application that uses machine learning or optimization to create personalized weekly schedules that students can use to prioritize their studying. It will include the following features:
- Scheduling based on due dates and workload
- Optimization or ML-based task allocation
- A web or desktop user interface
- AI‑assisted development guided by ROBOTS.md

## Running the Project (As of 04/20)
1. Clone the repo to GitHub desktop
2. Open with desired code editor (e.g., VScode)
3. Make sure python is installed and correct extensions are added
4. Click "Run" to generate output in the terminal
